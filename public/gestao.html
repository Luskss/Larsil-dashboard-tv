Serviços do Railway

Cada linha é um Project Token do Railway (Project Settings → Tokens). O rótulo aparece no card de status. Tokens já salvos ficam mascarados — deixe o campo em branco para manter o valor atual, ou digite um novo para substituir.

Páginas visíveis na navegação

Desmarque uma página para tirá-la da barra de bolinhas do rodapé (ela continua acessível pela URL). Arraste pela alça (⠿) para reordenar as bolinhas. Vale para todas as telas, inclusive a TV — ela acompanha em até 30 segundos, sem precisar recarregar.
